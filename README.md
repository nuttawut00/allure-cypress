# Allure Report

## Installation

```bash
npm install 
```
## Installation Allure
* For mac
```bash
$ brew install allure
```
* For Windows
```bash
scoop install allure
```


## Run Test
* Run Cypress UI
```bash
npx cypress open
```

## Run Cypress (Headless Mode) 
* Run Cypress (Headless Mode) 
```bash
npx cypress run
```
